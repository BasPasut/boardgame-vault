# BoardgameVault

Turn your physical board games into online sessions. Play with friends anywhere — no expensive physical sets required.

## Games

| Game | Status | Players |
|------|--------|---------|
| Shadows Over Thornwick | ✅ Available | 5–15 |
| Werewolf | 🔜 Coming Soon | 6–20 |
| Secret Hitler | 🔜 Coming Soon | 5–10 |

## Stack

- **Framework** — Next.js (App Router) + TypeScript
- **Styling** — Tailwind CSS v4
- **Real-time** — Supabase Realtime (planned)
- **i18n** — English + Thai
- **Deploy** — Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push to GitHub
2. Import repo on Vercel
3. Deploy — zero config needed
